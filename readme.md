bluphy
------

bluphy is a framework which enables web-based
remote interaction with server-local Bluetooth
devices through a master server-connected
Bluetooth dongle.

DEPENDENCIES/SYSREQ:

-Linux server

-Bluetooth

-PHP 5

-MySQL

-Python 3
